Installation

  1. Download and install Bo98's BPR Modder tool.
  2. Download the archive.
  3. Extract mod-manager.dll and imgui.dll to the game's root folder.
  4. Extract individual *.dll files to the game's mods folder previously created in step 1.
  1. Download the archive.
  2. Extract Game Files folder to the game's root folder.

Notes

  • Default hotkey for toggling menus is F7.
  • Default hotkey for toggling overlays is F8.
  • The hotkeys can be changed in mod-manager-config.yaml file.
  • Enabling ImGui's multiviewports feature will most likely cause FPS drops or crashes with other overlays.
  • Modding folder contains files tha you can use to integrate the pack with other mods.
