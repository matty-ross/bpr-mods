Installation

  1. Download and install Bo98's BPR Modder tool (if you haven't already).
  2. Download the archive.
  3. Extract mod-manager.dll and imgui.dll to the game's root folder.
  4. Extract individual mods/*.dll files to the game's mods folder previously created in step 1.
  1. Download the archive.
  2. Extract Game Files folder to the game's root folder.

Notes

  • Default hotkey for toggling menus is F7 (they aren't visible by default).
  • Default hotkey for toggling overlays is F8 (they are visible by default).
  • The hotkeys can be changed in mod-manager-config.yaml file.
  • Enabling ImGui's multiviewports allows you to drag the menus outside of the game's window (it may cause FPS drops or crashes).
  • Modding folder contains files that you can use to integrate the pack with other mods.
